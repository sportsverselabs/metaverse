Reference knowledge the system and agents can draw on — durable facts, guides,
playbooks, and documents about Sportsverse and its domain (sports).

Difference from `memory/`:

- **memory/** = dynamic, evolving facts agents learn and update over time.

Suggested future contents:
- Sportsverse brand guidelines (once defined)
- Domain knowledge / sport-specific references
- Operating playbooks and standard procedures
